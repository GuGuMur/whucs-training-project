\section{项目概述}

\subsection{项目定位}

随着数字化办公与学习的普及，个人和团队面临的文件管理挑战日益突出：文件分散在多个设备与平台、文档内容难以被快速检索、重复性文件处理任务消耗大量时间。传统的网盘解决了存储与同步问题，但在内容理解与任务自动化方面几乎空白；现有流行的
AI 问答工具能处理单轮对话，但缺乏对文件系统的持续感知能力和多步骤任务的编排能力。

本项目构建一个\textbf{以文件管理为基座、知识检索为大脑、智能体为交互中枢、团队协作为纽带}的一体化平台，旨在弥合上述三类工具的断层：让组织和个人在一个系统内完成文件的全生命周期管理、基于大模型的文档知识问答、以及多工具协作的智能任务编排。

\subsubsection{目标用户与应用场景}

\begin{itemize}
  \item \textbf{学生个人：} 管理课程资料、实验报告、论文笔记；用自然语言跨文档检索知识点；自动生成学习摘要
  \item \textbf{课程小组：} 团队文件夹共享资料；在线批注讨论；一键编排“收集成员报告 → 提取要点 → 汇总为团队成果”流程
  \item \textbf{科研团队（扩展场景）：} 文献库管理；跨论文知识问答；定期文献监控与自动摘要工作流
\end{itemize}

\subsubsection{核心差异化竞争力}

面对网盘、笔记软件、AI 问答工具三类现有产品，本平台的核心差异在于“三者合一”带来的协同效应：

\begin{enumerate}
  \item \textbf{不只是网盘——文件即知识。} 上传的 PDF、DOCX、PPTX 自动解析入库，图像经 PaddleOCR

        提取文字，所有内容向量化后进入检索索引。用户看到的不仅是文件列表，更是一个可被自然语言查询的知识库。答案标注引用来源，支持点击跳转至源文档原文段落。
  \item \textbf{不只是问答——流程即工具。} 用户不限于一问一答。通过可视化拖拽编辑器（Vue
        Flow），管理员预先定义工具库，用户自行组合工具节点、配置参数、连线定义数据流，构建专属的自动化工作流。例如“新文件上传 →
        自动摘要 →
        通知团队成员”三步流程，一次配置、持续运行。流程执行过程中 WebSocket 实时推送各节点的执行状态，支持单步调试。
  \item \textbf{不只是单机——协作即平台。} 内置 RBAC
        权限体系（超管/团队管理员/成员/访客四级角色），团队文件夹区别于个人文件夹，配合 WebSocket
        实时聊天、文件批注、活动动态流与 @
        通知，构成完整的团队协作闭环。知识库可设为私有/团队可见/全员可见，检索结果严格遵守权限边界。
\end{enumerate}

\subsection{用户故事}

\subsubsection{场景一：个人使用}

小明上传多份生物实验报告（PDF）与教师提供课件（PPTX），RAG解析PDF建立知识索引。小明在对话框输入“总结所有用到显微镜的实验步骤”，系统检索相关文档并生成对比摘要。

\subsubsection{场景二：团队协作}

课程小组创建团队空间“生物学实验”，邀请3位成员。成员上传各自的实验数据到团队文件夹，在文档上批注讨论。组长在工具流编排器中拖拽“文件扫描 → 内容提取 →
汇总报告”三个工具，配置参数后一键执行，Agent遍历团队文件夹中的所有文档生成汇总报告。

\section{功能模块详述}

\subsection{模块A：文件管理}

包含以下功能组：

\begin{itemize}
  \item \textbf{用户系统：} 注册、登录、JWT鉴权、个人信息管理
  \item \textbf{文件CRUD：} 上传（断点续传、分片上传）、下载、删除、重命名、移动、复制
  \item \textbf{文件夹管理：} 创建/删除/移动/嵌套，树形导航，面包屑路径
  \item \textbf{标签系统：} 手动标签创建/编辑/删除；自动标签同步，按标签筛选与聚合
  \item \textbf{全文搜索：} 文件名搜索（MySQL Fulltext）+ 标签联合筛选 + 文件类型过滤
  \item \textbf{版本控制：} 文件历史版本自动保存；版本列表查看；任意版本回滚/下载
  \item \textbf{文件分享：} 生成分享链接（支持密码保护 + 过期时间 + 下载次数限制）
  \item \textbf{数据安全：} 传输加密（HTTPS）；文件完整性校验（SHA256）；敏感文件标记
  \item \textbf{备份恢复：} 元数据定时SQL dump；文件存储异步同步；一键恢复
\end{itemize}

\subsection{模块B：RAG知识检索}

核心链路为：文档解析 → 分块 → 向量化 → 索引 → 语义检索 → 重排序 → 答案生成。

\begin{itemize}
  \item \textbf{文档解析：} 支持PDF/DOCX/PPTX/TXT/Markdown/CSV批量导入，提取纯文本 + 表格 +
        图片描述
  \item \textbf{知识库管理：} Web可视化界面：创建知识库、批量上传、分类标注、更新、归档、删除
  \item \textbf{文档分块：} 滑动窗口（chunk\_size=512, overlap=128），保留源文档ID + 段落位置元数据
  \item \textbf{向量化与索引：} BGE-M3 Embedding → FAISS索引；支持增量更新
  \item \textbf{语义检索：} Query Embedding → ANN Top-K召回 → Cross-Encoder
        Reranker精排
  \item \textbf{答案生成：} 检索片段 + 对话历史 + System Prompt → LLM流式生成，标注引用来源
  \item \textbf{多轮对话：} LangChain ConversationBufferWindowMemory，窗口大小可配置
  \item \textbf{引用溯源：} 回答中标记\texttt{[文档名 § 段落号]}，前端支持点击跳转至源文档预览
\end{itemize}

\subsection{模块C：可视化工具流编排系统}

管理员定义工具库，用户拖拽式构建工作流，实现ReAct模式的任务编排引擎。

\subsubsection{流程执行引擎}

\begin{itemize}
  \item \textbf{工具注册中心：} 统一工具Schema：名称、描述、参数定义（JSON Schema）、返回值类型。不少于6种工具
  \item \textbf{内置工具清单：}

        \texttt{file\_search}、\texttt{knowledge\_qa}、\texttt{image\_detect}、\texttt{file\_compare}、\texttt{team\_activity}
  \item \textbf{ReAct Agent：} “Thought → Action → Observation → ... →
        Answer”循环；LangGraph实现状态图
  \item \textbf{任务分解：} 复杂任务自动拆分为子任务DAG，确定执行顺序与依赖
  \item \textbf{执行调度：} 解析流程DAG → 拓扑排序 → 无依赖节点并行执行
  \item \textbf{容错与重试：} 单工具超时$\leq$30s；失败自动重试1次；重试失败触发降级策略
  \item \textbf{状态管理：} PENDING → RUNNING → SUCCESS / FAILED / CANCELLED
  \item \textbf{执行透明：}
        前端实时展示思考链，WebSocket实时推送每个节点的执行状态，保存每次触发记录（触发人、时间、参数、各节点输入/输出/耗时）
  \item \textbf{错误处理：} 节点失败策略可配置：终止/跳过/默认值
  \item \textbf{测试用例：} $\geq$10条，覆盖5类场景
\end{itemize}

\subsubsection{工具管理（管理员端）}

\begin{itemize}
  \item \textbf{工具注册：} 通过Web界面注册新工具：定义名称、描述、参数Schema（JSON
        Schema编辑器）、调用方式（API/Python函数/Shell命令）
  \item \textbf{工具分类：} 文件操作类、AI处理类、数据分析类、通知类、自定义类
  \item \textbf{工具测试：} 管理员可直接在界面测试调用，输入测试参数，查看返回结果
  \item \textbf{工具版本管理：} 支持版本号，更新后不影响已发布流程中引用的旧版本
  \item \textbf{工具市场：} 系统预置工具 + 管理员自定义工具，统一浏览和搜索
\end{itemize}

\subsubsection{流程编排（用户端）}

\begin{itemize}
  \item \textbf{可视化编辑器：} 基于Vue Flow实现拖拽式节点编辑器
  \item \textbf{节点类型：} 工具节点、条件节点（if/else分支）、循环节点、聚合节点、输出节点
  \item \textbf{参数绑定：} 支持上游节点输出 → 下游节点输入的参数引用
  \item \textbf{流程模板：} 预置模板（“新文件自动摘要”“团队周报生成”“图像批量标注”）
  \item \textbf{流程调试：} 单步执行模式，逐节点查看输入/输出
  \item \textbf{流程发布：} 支持手动触发或事件触发
\end{itemize}

\subsection{模块D：权限系统与团队协作}

\subsubsection{权限系统（RBAC）}

\begin{itemize}
  \item \textbf{RBAC模型：} 系统角色（超管/团队管理员/普通成员/访客）+ 自定义角色
  \item \textbf{资源级权限：} 对文件、文件夹、知识库、工具、流程模板分别控制：不可见/只读/读写/管理
  \item \textbf{权限继承：} 文件夹权限默认继承至子文件和子文件夹；可单独覆盖
  \item \textbf{工具使用权限：} 管理员可设置哪些角色/用户可以使用特定工具
  \item \textbf{知识库权限：} 知识库可设为私有/团队可见/全员可见
  \item \textbf{操作审计：} 记录关键操作日志，支持查询审计日志
  \item \textbf{权限检查中间件：} FastAPI中间件统一拦截请求
\end{itemize}

\subsubsection{团队协作}

\begin{itemize}
  \item \textbf{团队管理：} 创建团队、编辑团队信息、成员邀请、角色分配
  \item \textbf{团队文件夹：} 团队空间内创建文件夹树，区别于个人文件夹
  \item \textbf{实时聊天：} WebSocket驱动的团队群聊和1v1私聊；消息持久化存储
  \item \textbf{文件批注：} 在文件预览页面对文档/图片添加批注，成员可回复讨论
  \item \textbf{活动动态：} 团队动态流：成员操作以时间线展示
  \item \textbf{通知系统：} @提醒、批注通知、邀请通知、流程完成通知
\end{itemize}

  \item \textbf{文件分享：} 分享链接生成/密码校验/过期时间校验/下载次数上限/链接失效处理
  \item \textbf{数据完整性：} 上传前后 SHA256 一致性/分片上传合并后校验/断点续传状态恢复
\end{itemize}

\subsubsection{RAG 知识检索模块}

\begin{itemize}
  \item \textbf{文档解析：} PDF 纯文本提取/DOCX 表格保留/PPTX 图片描述/Markdown 格式保真/CSV
        字段解析/损坏文件容错，每种格式 $\geq$ 3 条用例
  \item \textbf{分块 Pipeline：} 中英文分块边界正确性/chunk\_size 与 overlap
        参数生效/空文档处理/超长文档（$>$10 万字）分块性能
  \item \textbf{向量索引：} BGE-M3 Embedding 维度校验（1024 维）/FAISS
        索引创建与持久化/增量添加/增量删除/批量操作原子性
  \item \textbf{语义检索：} Top-K 召回数量正确/ANN 近似精度（$\geq$ 95\% 与暴力搜索一致）/Reranker
        重排序后首位相关率/空查询处理
  \item \textbf{答案生成：} Prompt 模板变量注入正确性/LLM 流式输出分块/引用标注格式（\texttt{[文档名 §
              段落号]}）/生成超时兜底（$\leq$ 30s）
  \item \textbf{多轮对话：} ConversationBufferWindowMemory 窗口截断/跨轮引用一致性/会话过期清理
\end{itemize}

\subsubsection{智能体编排与工具流引擎}

\begin{itemize}
  \item \textbf{工具 Schema 校验：} 必填参数缺失报错/参数类型错误/返回值 Schema 校验/工具超时（$\leq$
        30s）自动中止
  \item \textbf{ReAct Agent：} Thought 解析/工具选择正确性（Mock LLM 返回固定
        Action）/Observation 注入 Prompt/Answer 终结条件触发
  \item \textbf{DAG 解析：} 拓扑排序正确性/环检测/孤立节点警告/单节点流程/线性流程/分支流程/并行分支
  \item \textbf{状态机：} PENDING $\to$ RUNNING  $\to$
        SUCCESS
        $\to$ FAILED $\to$ CANCELLED/非法状态转换拒绝
  \item \textbf{容错机制：} 单工具超时后重试次数/重试间隔（指数退避）/重试耗尽后降级策略触发/降级默认值注入
\end{itemize}

\subsection{安全测试}

\subsubsection{认证与授权}

\begin{itemize}
  \item \textbf{JWT 安全：} Token 过期后 API 返回 401；篡改签名后 API
        拒绝；伪造角色字段后权限中间件拦截；Refresh Token 单次使用后失效
  \item \textbf{越权访问：} 穷举测试：A 用户 Token 尝试访问 B 用户的文件/知识库/团队空间，全部预期返回 403
  \item \textbf{密码安全：} 密码存储使用 bcrypt/argon2 哈希（非明文/MD5）；登录失败 5 次后临时锁定（15
        分钟）；密码最小长度 8 位
\end{itemize}

\subsubsection{注入与输入校验}

\begin{itemize}
  \item \textbf{文件上传绕过：}
        \begin{itemize}
          \item 双扩展名（\texttt{shell.php.jpg}）——拒绝或仅识别真实 MIME
          \item MIME 头伪装（改 PNG 头为 PHP 代码）——MIME 嗅探校验
          \item 超大文件（$>$ 500 MB）——上传前拒绝，不消耗存储
          \item 空文件（0 字节）——允许但标记
          \item 路径穿越（\texttt{../../../etc/passwd}）——文件名规范化拒绝
        \end{itemize}
  \item \textbf{XSS 防护：} 以下字段输入
        \texttt{<script>alert(1)</script>}，验证前端渲染时被转义（非执行）：
        \begin{itemize}
          \item 文件名、文件夹名、标签名
          \item 聊天消息内容、批注内容
          \item 工具定义中的名称与描述
          \item 流程模板名称
        \end{itemize}
  \item \textbf{CSRF 防护：} 状态变更 API（创建/删除/修改）校验 SameSite Cookie 或 CSRF
        Token；跨域请求拒绝
\end{itemize}

\subsubsection{通信与基础设施安全}

\begin{itemize}
  \item \textbf{HTTPS：} 全站强制 HTTPS；HTTP 请求 301 重定向至 HTTPS；HSTS 头设置
  \item \textbf{WebSocket 安全：} 连接建立时验证 JWT Token；未认证 WebSocket 连接 5
        秒内关闭；跨团队频道消息隔离（A 团队无法监听 B 团队频道）；消息频率限制（单用户 $\leq$ 10 条/秒）
  \item \textbf{文件存储安全：} MinIO Bucket 非公开（禁止匿名列举）；文件下载链接一次性签名（有效期 $\leq$ 1
        小时）；敏感文件标记后额外加密存储
  \item \textbf{依赖安全：} CI 流水线集成 \texttt{npm audit}
        扫描已知漏洞；关键依赖锁定版本号
\end{itemize}
