02 / 12
背景与目标

为什么做这个平台

传统网盘只解决"存储与同步"，普通 AI 问答缺少对文件体系、团队权限和多步骤任务的持续感知。本平台以文件管理为基础、知识检索为核心、智能体为自动化入口。

五类用户
学生个人 / 课程小组 / 团队管理员 / 系统管理员 / 科研团队
六大业务目标
文件管理 · RAG 问答 · Agent 编排 · 工具流 · 团队协作 · 权限审计
七层架构
表现层 → API 层 → 领域服务 → AI 能力 → 异步任务 → 存储层
